# 📖 About

FinAssist AI is a next-generation **Agentic AI Financial Assistant** that helps users manage their finances through multiple specialized AI agents.

Unlike traditional chatbots, FinAssist AI routes every user query to the most suitable AI agent, enabling personalized financial guidance based on the user's financial profile, conversation memory, and Financial Digital Twin.

The platform combines **AI planning, voice interaction, privacy protection, financial analytics, and personalized recommendations** into a single intelligent financial ecosystem.

# 🚀 Workflow

1. User logs into the application.
2. User interacts through chat or voice.
3. Intent Router identifies the financial query.
4. The appropriate AI agent is selected.
5. User financial memory is retrieved.
6. Financial Digital Twin is updated.
7. AI CFP generates personalized recommendations.
8. Response is displayed to the user.
